默认:总页数12，显示页数5，初始化回调激活页绑定函数

$(".simplePaging").simplePaging();

总页数88，显示页数9,第一页显示为6，初始化跳转到7:

        $(".simplePaging2").simplePaging({
            allPage: 88,
            showPage: 9,
            startPage: 6,
            initPage: 7
        });

animationTime为50,回调显示页码:

        $(".simplePaging3").simplePaging({
            allPage: 88,
            showPage: 9,
            animateType: "animation",
            animationTime: 50,
            callBack: function (num) {
                $(".simplePaging3PageNum").text(num)
            }
        });

animateType为"jumpy":

        $(".simplePaging4").simplePaging({
            allPage: 88,
            showPage: 9,
            animateType: "jumpy",
            animationTime: 300,
            callBack: function (num) {
                $(".simplePaging4PageNum").text(num)
            }
        });

显示跳转按钮，每次页面首加载完毕不回调callBack，改变触发按键符号:

        $(".simplePaging5").simplePaging({
            allPage: 88,
            showPage: 9,
            startPage: 1,
            initPage: 1,
            initPageClick:false,
            showTurnTo: true,
            first: "first",
            last: "last",
            prev: "<",
            next: ">",
            animateType: "normal",
            animationTime: 300,
            callBack: function (num) {
                $(".simplePaging5PageNum").append(num+"    ")
            }
        });

文档说明：
参数名 参数类型 可选参数 说明
allPage number 自定义数字 默认值：12。总页数：页码器总共多少页。
showPage number 自定义数字 默认值：5。显示页数：页码器显示几页。
startPage number 自定义数字 默认值：1。首页页码数字。
initPage number 自定义数字 默认值：1。加载完毕自动跳转到第n页：此值小于startPage则跳转首页，反之跳转尾页。
initPageClick boolean true或false 默认值：true(回调)。每次页面加载完毕后，是否触发initPage页的绑定事件,false不回调。
first string 自定义字符串 默认值："首页"。首页显示字符。
last string 自定义字符串 默认值："尾页"。尾页显示字符。
prev string 自定义字符串 默认值："«"。上一页显示字符。
next string 自定义字符串 默认值："»"。下一页显示字符。
showTurnTo boolean true或false 默认值：false(不显示)。是否显示“跳转”项，true时显示。为true时，页码输入框输入完毕，按下回车键即可立即执行跳转。输入页码小于startPage跳转首页，反之跳至尾页。
animateType string "animation"
"jumpy"
"fast"
"normal"
"slow"
"verySlow" 		默认值："animation"。动画过渡模式。
立即，对应animationTime=0
100ms，对应animationTime=100
200ms，对应animationTime=200
400ms，对应animationTime=400
800ms，对应animationTime=800
animationTime number 自定义数字 默认值：300。仅为animateType为animation时生效，设置动画过渡时间(ms)。
callBack function 自定义方法 默认值：打印页码。用于回调函数扩展方法。

默认参数

        allPage: 12,//总页数
        showPage: 5,//显示页数
        startPage: 1,//第一页页码数字
        initPage: 1,//加载完毕自动跳转到第n页(初始化激活页)
        initPageClick:true,//每次页面加载完毕后，是否触发initPage页的绑定事件
        first: "首页",//首页显示字符
        last: "尾页",//尾页显示字符
        prev: "«",//上一页显示字符
        next: "»",//下一页显示字符
        showTurnTo: false,//是否显示跳转按钮，true显示，false不显示
        animateType: "animation",//过渡模式：动画“animation”、跳动“jumpy”、快速移动“fast”、正常速度移动“normal”、缓慢的速度移动“slow”、异常缓慢的速度移动“verySlow”
        animationTime: 300,//animateType为animation时，动画过渡时间(ms)
        callBack: function (num) {
            console.log(num)
        }
